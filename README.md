# Data Visualization Dashboard

A responsive dashboard built using **HTML, CSS, JavaScript**, and **Chart.js** to visualize sample data.

## Features
- Bar chart for monthly sales
- Line chart for website traffic
- Pie chart for device usage
- Responsive layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js
